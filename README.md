
  # Couple Financial Dashboard

  This is a code bundle for Couple Financial Dashboard. The original project is available at https://www.figma.com/design/FbIT1BYFMRonnYvyhHlyqC/Couple-Financial-Dashboard.

  ## Running the code

  Run `npm i` to install the dependencies.

  ## Environment variables

  Copy `.env.example` to `.env.local` and fill in your Supabase values:

  - `VITE_SUPABASE_URL`
  - `VITE_SUPABASE_ANON_KEY`

  Keep `.env.local` out of git. It is already ignored by `.gitignore`.

  Run `npm run dev` to start the development server.
