# Code of Conduct

## Our Pledge

Participantes deste projeto devem colaborar com respeito, empatia e profissionalismo.

## Expected Behavior

- Use linguagem acolhedora e objetiva.
- Respeite diferencas de experiencia, contexto e opiniao.
- Foque feedback em codigo, produto e impacto.
- Aceite correcoes com boa fe.

## Unacceptable Behavior

- Assedio, intimidacao ou ataques pessoais.
- Comentarios discriminatorios.
- Exposicao publica de informacoes privadas.
- Conduta que torne o ambiente hostil.

## Enforcement

O mantenedor pode remover comentarios, rejeitar contribuicoes ou bloquear participacao quando necessario para proteger o projeto.
